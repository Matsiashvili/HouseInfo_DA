# Data Analysis - Homework 2

## About
This is the second homework assignment for the Data Analysis course.
The project explores Housing dataset from sklearn,
performing exploratory data analysis (EDA) on housing data.

## Dataset
The dataset contains housing block information.
Each block includes features like income, house age, rooms, population
and location, with the target variable being the median house price.

## Tasks
- Task 1: Find min, max, mean and std for each column using numpy
- Task 2: Check correlation between HouseAge and Price
- Task 3: Check correlation between AveRooms and Price
- Task 4: Check correlation between AveBedrms and Price
- Task 5: Remove highly correlated columns (except Price)
- Task 6: Define a function and compare distributions for HouseAge, AveRooms and Population
- Task 7: Compare global distributions of Population and AveRooms and conclusion on which column is most important for price prediction

## Libraries Used
- pandas
- numpy
- matplotlib
- scikit-learn
